# Lemay.ai-interview-part3 - Exploratory data analysis
 
## Exploratory data analysis of Rotten Tomatoes review data

The Rotten Tomatoes review dataset is a commonly used dataset for sentiment analysis due to its large amount of textual data that can be analyzed using natural language processing techniques. Additionally, performing exploratory data analysis on a textual dataset can be a challenging task, making it crucial to gain insights into the data before proceeding with modeling and prediction.
